# ClassAI 프로그램 상세 사용 설명서 (Program Instruction)

## 1. 개요 (Introduction)
**ClassAI**는 맨큐의 경제학(Mankiw's Principles of Economics)을 기반으로 한 **AI 기반 교육 플랫폼(AI Learning Platform)**입니다.
교수자와 학습자 모두에게 가장 효율적인 학습 경험을 제공하기 위해, 방대한 교과서 내용을 36개의 챕터로 구조화하고, AI의 강력한 분석 및 생성 능력을 결합했습니다.

---

## 2. 주요 기능 (Core Features)

### 2.1. 🔍 교과서 자료 찾기 (Textbook Search)
*   **기능:** 36개 챕터의 핵심 개념, 요약, 키워드를 빠르게 탐색합니다.
*   **특징:**
    *   **검색:** 키워드와 챕터명을 입력하면 관련된 내용을 실시간으로 찾아줍니다.
    *   **요약 뷰:** 각 챕터의 주요 내용을 한눈에 볼 수 있는 요약 카드를 제공합니다.
    *   **AI 연동:** 단순 텍스트 매칭을 넘어, 문맥에 맞는 내용을 찾아주는 RAG(Retrieval-Augmented Generation) 기반 검색.

### 2.2. 📝 연습문제 풀기 (Practice)
*   **기능:** 각 챕터별 맞춤형 퀴즈와 문제를 생성하여 학습 이해도를 점검합니다.
*   **특징:**
    *   **문제 유형:** 객관식, O/X 퀴즈, 단답형, 서술형 등 다양한 문제 유형 지원.
    *   **난이도 조절:** 학습자의 수준(초급, 중급, 고급)에 맞춰 문제 난이도 선택 가능.
    *   **즉시 채점:** 문제를 풀고 제출하면 즉시 정답 및 상세 해설 제공.

### 2.3. 💯 시험문제 내기 (Exam Generation)
*   **기능:** 교수자(Instructor)를 위해 실제 시험과 유사한 모의고사를 자동으로 생성합니다.
*   **특징:**
    *   **범위 설정:** 중간고사(Ch.1-17), 기말고사(Ch.18-36) 또는 사용자 지정 범위 선택 가능.
    *   **커스터마이징:** 문제 수(5~25문항), 유형 비율, 시험 시간 설정 가능.
    *   **인쇄 및 다운로드:** 생성된 시험지를 PDF/Word 형식으로 다운로드하거나 웹에서 바로 인쇄 가능.

### 2.4. 🎓 강의자료 만들기 (Lecture Materials)
*   **기능:** 수업 준비 시간을 획기적으로 줄여주는 맞춤형 강의 자료 생성.
*   **자료 구성:**
    *   **수업 개요 및 목표:** 50분/75분 수업에 맞춘 상세 시간표(Lesson Plan).
    *   **강의 노트:** 핵심 설명 포인트와 예시가 포함된 스크립트.
    *   **PPT 슬라이드 개요:** 각 슬라이드에 들어갈 제목과 본문 내용 구조화.
    *   **토론 주제:** 수업 참여를 유도하는 심화 질문 리스트.

### 2.5. 📑 보고서 작성 (AI Report Assistant)
*   **기능:** 경제학 주제에 대한 심층적인 분석 보고서 초안을 작성해줍니다.
*   **특징:**
    *   **주제 추천:** 챕터를 선택하면 관련된 시사 이슈나 논문 주제를 제안(Suggest).
    *   **체계적 구조:** 서론-본론(현황/이론적 분석/사례)-결론의 논리적 구조 자동 생성.
    *   **참고문헌:** 신뢰할 수 있는 출처(한국은행, 통계청 등)를 포함한 참고문헌 목록 제공.

### 2.6. 🤖 AI 튜터 (AI Interactive Tutor)
*   **기능:** 언제든 실시간으로 궁금한 점을 해소할 수 있는 대화형 챗봇.
*   **접근성:** 화면 우측 하단의 플로팅 위젯을 통해 어느 페이지에서든 호출 가능.
*   **동작:** 사용자의 질문을 인식하여 관련된 교과서 내용을 기반으로 정확한 답변 제공.

## 3. 기술 스택 (Tech Stack)

*   **Frontend:** HTML5, CSS3 (Advanced Variables, Flexbox/Grid), Vanilla JavaScript (ES6+).
*   **Styling:**
    *   **Theme System:** 사용자 환경(OS Dark Mode)을 감지하는 반응형 다크 모드.
    *   **Glassmorphism:** 현대적인 UI 트렌드를 반영한 반투명 유리 효과.
    *   **FontAwesome:** 직관적인 아이콘 시스템.
*   **Backend Interface (Simulated):**
    *   `RAG System`: 교과서 텍스트 임베딩을 통한 정확한 정보 검색.
    *   `LocalStorage`: 사용자 테마 및 설정 저장.
    *   `Mock API`: 빠르고 안정적인 데모 환경을 위한 데이터 시뮬레이션.

## 4. 사용 방법 (User Guide)

1.  **시작하기:** 메인 페이지(`index.html`)에서 원하는 학습 모드를 선택합니다. (Search, Practice, Exam 등)
2.  **로그인:** 별도의 로그인 없이 게스트 모드로 모든 기능을 체험할 수 있습니다.
3.  **검색 및 학습:**
    *   `Search` 메뉴로 이동하여 관심 있는 키워드(예: "수요공급")를 검색합니다.
    *   우측 패널에 표시된 챕터 요약을 읽고 `연습문제 풀기` 버튼을 눌러 학습을 이어갑니다.
4.  **자료 생성:**
    *   `Lecture` 또는 `Report` 메뉴에서 챕터를 선택하고 `생성하기` 버튼을 누릅니다.
    *   결과물을 확인한 후 `다운로드` 버튼을 통해 파일로 저장합니다.
5.  **AI 질문:** 화면 우측 하단의 로봇 아이콘을 클릭하여 AI 튜터와 대화하며 궁금증을 해결합니다.
